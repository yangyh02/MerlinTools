# 梅林初号机 - AFK Journey 自动化脚本

一个用于 AFK Journey（剑与远征：启程）的自动化脚本工具。

## 功能列表

| 功能 | 说明 |
|------|------|
| 登录 | 自动登录游戏 |
| 仓库管理 | 自动识别并记录英雄信息 |
| 邮件 | 自动领取邮件奖励 |
| 好友奖励 | 自动领取好友赠送奖励 |
| 普竞 | 自动完成普竞挑战 |
| 迷梦之域 | 自动挑战迷梦之域（可设置次数） |
| 女神塔 | 自动挑战女神塔 |
| 挂机奖励 | 自动领取挂机奖励（可设置付费次数） |
| 商城 | 自动购买友谊商城打折物品 |
| 日常任务 | 自动领取日常任务和公会任务奖励 |
| 爬塔 | 自动爬塔 |
| 幻灵推图 | 自动幻灵推图 |
| 推图 | 自动推图 |

## 运行环境

- **操作系统**: Windows 10/11
- **游戏分辨率**: 1600*900
- **游戏模式**: 横屏

## 使用方法

### 1. 准备工作

右侧release下载最新版本

确保文件夹结构如下：

```
梅林初号机/
├── goldenhandmaidens.exe
└── shared/              (需要手动创建此空文件夹)
```

### 2. 运行程序

1. 启动游戏并进入主界面
2. 以管理员身份运行 `goldenhandmaidens.exe`
3. 在界面中勾选需要执行的功能
4. 点击「开始运行」

### 3. 配置说明
- **登录**: 处理开始游戏到登录后的月卡和礼包界面
- **仓库管理**: 推图以及爬塔需要最新仓库信息支持，如果角色更新请再运行一次以更新warehouse_heroes.txt，升级标志可能导致共鸣之手的巅峰+以下角色识别错误，部分角色皮肤未收录可能识别错误，请尽量避免
- **普竞**: 挑战最左边玩家，默认使用完所有免费次数
- **邮件**: 自动领取全部邮件，需要囤积邮件请手动关闭
- **迷梦之域**: 可设置挑战次数
- **女神塔**: 默认用完普通徽记，不会使用精致徽记
- **每日任务**: 需要卡公会币上限请暂时关闭此功能
- **挂机奖励**: 可设置付费购买次数（0-2次）
- **推图和爬塔**: 推图会跳过手动作业，并且使用练度符合的阵容，其中练度只分为巅峰+及以上和巅峰+以下两种


## 文件说明

| 文件/文件夹 | 说明 |
|-------------|------|
| `goldenhandmaidens.exe` | 主程序 |
| `shared/` | 程序运行时需要的文件夹（首次使用需手动创建） |
| `game_bot_config.json` | 配置文件（程序自动生成） |
| `warehouse_heroes.txt` | 仓库英雄记录（程序自动生成） |

## 注意事项

1. **首次运行**：请先手动创建 `shared` 空文件夹
2. **运行期间**：请勿移动鼠标或操作游戏窗口
3. **模板匹配**：如果识别失败，请检查游戏分辨率和界面是否正确
4. **停止运行**：点击「停止运行」按钮或关闭窗口

## 常见问题

### Q: 程序启动后没有反应？
A: 请确保 `shared` 文件夹已创建，且游戏已在运行。

### Q: 识别不到按钮？
A: 请检查游戏分辨率是否正确，界面是否与模板匹配。

### Q: 如何停止正在运行的任务？
A: 点击界面上的「停止运行」按钮。
